Mario Kart (v3)
===========

![Screenshot](screenshot.png)

Small homebrew game written in Lua for PSP 


##What's this ?
A minimalistic kart racing simulation.

##Notes
Compatible with 1.50, 3.XX and 4.XX Kernels. <br/>
Should work on PSP Slim and Fat (Uses *LPHM 7*).

##Installation Path
ms0:/PSP/GAME/
